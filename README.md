## Project Features

- Built with pure PHP using OOP (no frameworks)

- Uses PDO (PHP Data Objects) for secure database interaction with prepared statements and parameter binding

- Three database tables: authors, categories, and quotes with relational joins

- Full CRUD (Create, Read, Update, Delete) REST API endpoints for each resource

- Input sanitization via htmlspecialchars and strip_tags on all write operations

- Relational filtering endpoints such as querying quotes by author, category, or both combined

- Consistent JSON responses across all endpoints with descriptive error messaging
