## Project Features

- Built with pure PHP using OOP (no frameworks)

- PHP OOP PDO with full CRUD REST API endpoints for each resource

- Three database tables: authors, categories, and quotes with relational joins. Relational filtering endpoints such as querying quotes by author, category, or both combined. Consistent JSON responses across all endpoints with descriptive error messaging

- PostgreSQL managed database on Render

- Dockerized application with a clean Dockerfile

- GitHub source control. Render auto-deploying every push
